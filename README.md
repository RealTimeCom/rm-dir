## rm-dir

**Remove directory and all its contents, sync or async**

```sh
$ npm install rm-dir
```
#### Example
```js
const rmdir = require('rm-dir');

rmdir('/dir/path/s-name'); // SYNC

rmdir('/dir/path/a-name', console.log); // ASYNC
```
